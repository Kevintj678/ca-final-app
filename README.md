# CA Final Study Planner

A study tracking app for CA Final students — Nov 2026 & May 2027 batches.

## Features
- Student registration & login
- Daily study session logging
- Weekly timetable planner
- Chapter progress tracker (Paper 4 & Paper 5)
- Admin panel to monitor all students
- PWA — installable on mobile

## Setup
Set these environment variables in Vercel:
- `REACT_APP_SUPABASE_URL` = your Supabase project URL
- `REACT_APP_SUPABASE_KEY` = your Supabase publishable key
